<h1>IONIC</h1>


## EJECUTAR

Instalar modulos
```bash
npm i
```

Iniciar servicio
```bash
ionic serve
```

Compilar local (environments.ts)
```bash
ionic build
```

Compilar local (environments.prod.ts)
```bash
ionic build --prod
```


## EJEMPLO DE IMPLEMENTACIÓN EN XAMPP
La compilación crea la carpeta: www
Copiar el contenido de www
Pegar en C:\xampp\htdocs
